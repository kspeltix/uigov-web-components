# Component Integratie

Technische documentatie is te vinden in Storybook waarvan je - geversioneerd - een referentie vindt vanuit de
[UIGOV-Pages](https://milieuinfo.github.io/uigov-pages/)

We zijn storybook documentatie van de componenten geleidelijk aan het verbeteren, per individueel component is er een "Docs" pagina die meer detail heeft over het gebruik van het component.

In de verbeterde versie vermelden we doorgaans:
- Beschrijving van het component
- Voorbeeld:
    - welke imports benodigd
    - hoe je de template integreert
- Default gebruik voorbeeld
- Configuratie
- Varianten
- Referenties met onder meer links naar de vorige versies van het component en afgeleide van Digitaal Vlaanderen
