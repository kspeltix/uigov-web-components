# Range

Het range component kan gebruikt worden om een bereik te selecteren binnen een minimum- en maximumwaarde. Het kan ook gebruikt worden als filter.

## Voorbeeld

```js
import { VlRangeComponent } from '@domg-wc/components';
```

```html
<vl-range></vl-range>
```
